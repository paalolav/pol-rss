# SPFx Enterprise Skills (project-local)

These four skill files provide Claude Code with SPFx-specific guardrails for
work in this repo. They are loaded automatically by Claude Code when a session
runs from this project root.

| Skill | Use for |
|---|---|
| `spfx-image-and-media-optimization` | Image rendering, lazy-loading, CLS, alt-text |
| `spfx-property-pane-reactivity` | Reactive vs non-reactive property pane decisions |
| `spfx-accessibility-and-content-quality` | Keyboard, focus, aria, error/empty/loading copy |
| `spfx-enterprise-code-and-performance` | Service/hook/component structure, bundle size, caching |

Source: <https://github.com/sudharsank/spfx-enterprise-skills>
